# LLM Prompting for Multi-Class Sexism Detection
Zero-shot and few-shot classification with open-source Large Language Models.

---

##  Overview
This project explores the use of open-source LLMs for multi-class sexism detection (EDOS Task B), using structured prompting strategies in zero-shot and few-shot settings.

The focus is on prompt design, automated response parsing, and systematic evaluation.

---

##  Tasks
- Multi-class sexism classification
- Zero-shot and few-shot prompting
- Structured output generation
- Automatic evaluation and error analysis

---

##  Technologies
- Python  
- HuggingFace Transformers  
- Open-source LLMs  
- pandas, scikit-learn  

---

##  Methodology
- Prompt engineering for multi-class classification  
- Few-shot example construction  
- Automatic response parsing  
- Evaluation with macro F1 and confusion matrices  

---

##  Results
The project compares zero-shot and few-shot setups, highlighting strengths and limitations of prompt-based classification for sensitive NLP tasks.
